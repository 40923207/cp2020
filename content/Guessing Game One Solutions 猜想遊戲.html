Random Numbers (and Modules)隨機數字(和模塊)

This is your first exposure to using Python code that somebody else wrote. In Python, these formally-distributed code packages are called modules. The thing we want from a mnodule in this exercise is the ability to generate random numbers. This comes from the random numbers.This comes from the random module.

這是您第一次使用別人編寫的Python代碼。在Python中，這些正式分發的代碼包稱為模塊。在本練習中，我們希望從小結中獲得的東西是產生隨機數的能力。這來自隨機數，這來自隨機模塊。

To use a module, at the top of your file, type使用模塊,在文件頂部鍵入

import random


This means you are allowing your Python program to use a mod ule called random in the rest of your code.

這意味着你允許你的蟒蛇程序在剩下的代碼中使用一個稱爲隨機模態。

To use it (and generate a random integer), now type:

要使用它(併產生一個隨機整數),現在鍵入:

a = random.randint(2, 6)


Once you run this program, the variable a will have a random integer that the computer made for you, bettween 2 and 6 (including 2 and 6). The specific documentation for this method is here.

一旦運行這個程序,變量a將有一個計算機爲你製造的隨機整數,bett2之間。及6個(包括2個及6宗)。 這個方法的具體文件在這裏。

There are many ways you can generate random numbers - integers, decimals, and much more. The Python

documentation has much more detailed information about what is possible from the random module.

有很多方法可以產生隨機數,整數,十進制等等。 蟒蛇

文檔中有關隨機模塊可能包含的更詳細的資料。

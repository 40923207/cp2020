Guessing Game One Solutions 猜想遊戲解決方案

Exercise 9練習九

Generate a random number between 1 and 9 (including 1 and 9). Ask the user to guess the number, then tell them whether they guessed too lorw, too high, or exactly right. (_Hint: remember to use the user input lessons from the very first exercise

在1到9之間產生隨機數(包括1到9)。 請用戶猜數字,然後告訴用戶他們是否猜得過高,過高或完全正確。(_Hint:記住使用輸入法。從第一次練習開始

Extras:附加功能：

ㆍKeep the game going until the user types "exit" 保持遊戲進行到用戶輸入"退出"爲止

ㆍKeep track of how many guesses the user has taken, and when the game ends, print this out.記錄用戶猜了多少次,遊戲結束後,把這個打印出來。

Sample solution樣本溶液

Great example of a solution including both extras!

包括兩個附加的解決方案的好例子!

 

import random

number = random.randint(1,9)
guess = 0
count = 0


while guess != number and guess != "exit":
    guess = input("What's your guess?")
    
    if guess == "exit":
        break
    
    guess = int(guess)
    count += 1
    
    if guess < number:
        print("Too low!")
    elif guess > number:
        print("Too high!")
    else:
        print("You got it!")
        print("And it only took you",count,"tries!")

import random

rd = random.randint(1,9)

    if guess == "exit":
        break

    guess = int(guess)
    c += 1

    if guess < rd:
        print("Too low")
    elif guess > rd:
        print("Too high")
